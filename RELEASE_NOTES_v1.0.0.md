# Release v1.0.0

Initial release of Color Palette Maker (React)

## Features

- Extract color palettes from images using K-means clustering
- Detect image regions with OpenCV
- Manual color sampling
- Export palettes as JSON
- Light/dark theme support
- Full test coverage (87%+ statements, 81%+ branches)

## Installation

```bash
npm install
cd client && npm install && cd ..
npm run dev
```
